# 🚀 Netlify Deployment Guide

## Quick Deploy (30 seconds)

```bash
npm run build
# Then drag dist/ to Netlify
```

## Step by Step

### 1. Build Production Build
```bash
npm run build
```

### 2. Deploy to Netlify
- Visit netlify.com
- Drag `dist/` folder
- Your app is live!

### 3. Custom Domain
- Go to Site settings
- Add custom domain
- Update DNS records

## Environment Variables
Add in Netlify dashboard settings.
